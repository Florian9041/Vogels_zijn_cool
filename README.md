# Vogels_zijn_supercool
Er zijn honderden soorten vogels in België. Je vindt ze in allerlei kleuren en vormen.
Het is niet makkelijk om alle vogels te zien, je moet hiervoor het hele land rondrijden en veel geluk hebben.
Vogels kijken vraagt dus veel geduld. Naar een gebied gaan om een bepaalde vogel te spotten noemen we "twitchen".
In België vind je ze vooral in de haven van Zeebrugge en de Fonteintjes, gewapend met een telelens voor de mooiste foto van een zeldzame vogel.

**De Klapekster**

De klapekster is een heel mooie zeldzame vogel.
Ik heb hem zelf nog maar twee keer gezien.
